# Bus Management App

Full-stack application for managing bus routes, stops, and live vehicle tracking.  
This project includes both the frontend and backend, with a modular structure for scalability and maintainability.

## Features (in progress)

- Live map displaying buses on multiple routes  
- Admin panel for managing buses, stops, and routes  
- Responsive layout and interactive map UX  

---

## Installation 

1. Clone the repository  
2. Install dependencies in [frontend](https://github.com/LouqmaneB/Bus-Management-System/tree/main/frontend) and [backend](https://github.com/LouqmaneB/Bus-Management-System/tree/main/backtend) folders  
3. Configure environment variables
4. Start backend and frontend
