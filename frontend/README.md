# Frontend – Bus Management App

This frontend is the user-facing web application of the Bus Management App.
It focuses on map-based visualization, route display, and a responsive user interface.
The frontend communicates with the backend REST API.

## Implemented Features (Current Stage)

- Main application layout and page structure
- Interactive map using **MapLibre**
- Rendering multiple bus routes with different colors
- Display of routes and stops on the map
- Improved map usability:
  - Card-based layout
  - Fullscreen mode
  - Controlled scroll zoom
- Integration with backend API for routes and stops
- Responsive design for different screen sizes

## Running the Frontend

1. Install dependencies  
   ```bash
   npm install
    ```
2. Start the development server  
   ```bash
   npm run dev
    ```
3. Open in browser  
   ```bash
   http://localhost:3000
    ```
